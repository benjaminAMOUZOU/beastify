# Beastify
Ajoute une icône à la barre d'outil pour une action navigateur. Cliquez sur le bouton pour choisir une bête. Le contenu de l'onglet actif sera alors remplacé par une image de la bête choisie. 

## 1. Comment installer
- Dans la barre de navigation de firefox, taper `about:debugging` ; 
- Aller dans l'ongle `Ce Firefox` ;
- Cliquer sur le bouton `Charger un module complémentaire temporaire` ;
- Choississez le fichier `manifest.json` ;
- Cliquer sur le bouton charger.

## 2. Comment tester l'extension
Une icone s'affichera dans la barre d'extension en haut à droite. Cliquer dessus pour choisir l'image de fond souhaitée.
